# office-28circle-notes

一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何具体网站或域名，仅作为页面资源存储与分发的基础空间。

## 项目简介

`office-28circle-notes` 主要用于存放各类独立的 HTML 文档，方便归档管理及直接通过 GitHub Pages 或其他静态服务进行访问。  
每个页面均为独立的完整 HTML 文件，彼此之间无依赖关系，可单独打开使用。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 存放主要 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放所有归档的 HTML 页面，每个文件独立成页。
- `assets/` 目录用于存放页面引用的公共资源，可根据需要组织子目录。

## 页面归档说明

- 所有 HTML 页面均为静态文件，不涉及后端服务或数据库。
- 页面内容由维护者手动添加或更新，无自动同步机制。
- 仓库不保证页面的实时性或长期可用性，仅作为存档用途。

## 维护说明

- 欢迎提交 Pull Request 添加或修改页面，但请确保页面内容不违反相关法律法规及平台政策。
- 页面命名建议使用有意义的英文或拼音，避免特殊字符。
- 若页面引用外部资源（如字体、脚本），建议优先使用相对路径或自托管版本，以减少外部依赖。
- 本仓库不提供页面内容的审核或担保，使用者需自行判断页面适用性。
